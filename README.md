# Dance Jam course site

This website was developed with [Quarto](https://quarto.org) and hosted on GitHub. Template code found at: https://pakillo.github.io

How to update the website:
1. Start a codespace
2. Go to Terminal and execute `pip install quarto-cli`
3. Run command `quarto preview` to view the website on a local port
4. Make changes and commit to see them reflected in the deployed site.

How to host this site:
1. Go to `Settings` > `Pages` (under Code and Animation on left) > select item from `Branch` drop down > Save
2. Go to `Code` > `About` (right side) > click gear > select `Use your GitHub Pages website` for url
